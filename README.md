# Notes App

Приложение для создания заметок "Notes App".

Возможности:
- Создание папок (категорий) по одной общей теме
- Сортировка папок
- Создание заметки в выбранной категории
- Поиск необходимой заметки
- Сортировка заметок
- Удаление заметки
- Просмотр информации о заметке
